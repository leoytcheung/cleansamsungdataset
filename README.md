README
======

This repo contains a script that produce a tidy dataset of the accelerometer and gyroscope data on human activities from the SmartLab of the Università degli Studi di Genova.

The repo contains the following:

1. a script called "run_analysis.R" that runs on R;

2. a codebook that explains the cleaned dataset, the original data and the variables in the cleaned dataset.


Before running the script, you need to:

1. download the zip file from the following address: 
	https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 

2. unzip the downloaded file and save the folder "UCI HAR Dataset" and its files inside to the working directory of your R;
